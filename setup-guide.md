# Setup Guide

This guide explains how to run Flowise locally.

## Step 1: Install Docker

Download Docker Desktop:

https://www.docker.com/products/docker-desktop/

## Step 2: Enable WSL2

Install WSL2 in PowerShell:

wsl --install

## Step 3: Start Flowise

Run the following command:

docker compose up -d

## Step 4: Open Flowise

Go to:

http://localhost:3000
